# 回调函数

在实际的项目编程中，我们有时候需要一个在一个函数内调用我们指定的函数，即将一个函数作为参数传入另外一个函数，然后在那个函数内调用它。

如以下伪代码所示：

```cpp
void CallFunction(Function)
{
    ... // Do something
        
	Function();
    
    ... // Do something
}

void SayHello() { }

void SayGoodBye() { }

void main()
{
    CallFunction(SayHello);
    CallFunction(SayGoodbye);
}
```
